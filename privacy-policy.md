# Privacy Policy – AI Tenant Insights

Effective Date: March 2026

AI Tenant Insights provides AI-powered feedback analysis for property managers.

## Information We Collect
- Tenant identifiers (UUID)
- Feedback messages submitted for analysis
- AI-generated structured analysis results

We do not collect:
- Payment information
- Personal authentication passwords
- Sensitive personal identification numbers

## How We Use Information
Collected data is used solely to:
- Analyze tenant feedback
- Provide structured AI insights
- Improve service reliability

## Data Storage
Data is stored securely using Supabase PostgreSQL infrastructure.

## Data Sharing
We do not sell or share user data with third parties.

## Security
We use secure API-based authentication and encrypted database connections.
